При введении параметров программы, следует руководствоваться следующим.

Для добавления нового платежа нужно ввести параметры следующего типа:
1) 	payment_number payment_date recipient_id amount
2) "payment_number payment_date recipient_id amount"
Для добавления сразу нескольких новых платежей:
3) "payment_number1 payment_date1 recipient_id1 amount1" "payment_number2 payment_date2 recipient_id2 amount2" ...

Даты payment_date вводятся в следующих форматах:
yyyy-mm-dd
yyyy/mm/dd

Для ввода нового контрагента нужно ввести параметры следующего типа:
4) 	id entity_name
Наименования контрагентов, состоящие из нескольких слов необходимо заключать в двойные кавычки.


---
При запуске программы без параметров, будет выведен текущий список платежей
Для вывода списка контрагентов и их ID следует ввести "-contractors"

---
Данная программа использует перекомпилированные запросы для добавления записей.
